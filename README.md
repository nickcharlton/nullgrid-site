# nullgrid-site

This is the site for [nullgrid][].

## Usage

It's a bare-bones [Sinatra][] application. Run `bundle install` and
then:

```
foreman start
```

## Testing

It uses [RSpec][] for testing:

```
bundle exec rspec
```
